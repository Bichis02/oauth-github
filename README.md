# Práctica: Implementación segura de OAuth 2.0 con GitHub

![Logo de la app](logo.svg)

Este repositorio contiene los archivos de la práctica de **Autenticación OAuth 2.0**.

## Contenido
- `oauth_login.php`.
- `oauth_callback.php`.
- Versión vulnerable y versión mitigada.

## Objetivo
Mostrar cómo una implementación incorrecta de OAuth puede permitir ataques como **CSRF** o **reutilización del code**, y cómo mitigarlos validando `state` y ocultando el `authorization code`.
